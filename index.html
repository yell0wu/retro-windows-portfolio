About Dennis

This is my portfolio.


Hi! My name is Dennis Wu, and this is my portfolio and final project for CS103.


CS103 is taught by Professor Vahid Azadeh-Ranjbar and is an introductory course about web design, web application programming, and general web development.


Brief Introduction

I'm a computer science student and in my senior year at Boston University. In the future, I hope to be doing work as either a software developer or a full-stack web developer.
